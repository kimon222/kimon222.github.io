About Me

Hi, my name is Kimon! I'm a frontend developer with a focus on AI. I recently earned my computer science degree from San Francisco State University. I have experience working across organizations such as Dev/Mission, Uber, and Mission Bit. During my frontend fellowship with Uber, I created a digital sign-in web application (SignMe) with a team, using React.js, HTML5, and CSS3. SignMe is currently being implemented across 5+ locations.

Furthermore, I was recently inspired by Udemy's course on developing LLM powered applications using LangChain, and created my own project called "FindMySong" which finds song titles and artist names from user-entered song lyrics. FindMySong uses HTML5 and CSS3 for the user interface, and LangChain and Python to find song data using modern LLMs.
